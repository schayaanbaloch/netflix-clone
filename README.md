# Netflix Clone

A Netflix-style web application built as a frontend + backend project.

## Tech Stack
- React (Vite)
- Pure CSS
- Node.js
- Express
- TMDB API

## Features
- Dynamic movie listings from TMDB
- Trailer modal with YouTube embed
- Live search functionality
- Hover animations on movie cards
- Backend API to protect API keys

## How to Run
### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm run dev
